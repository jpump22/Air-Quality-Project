# Raspberry Pi Air Quality Monitoring System
Air quality monitoring system developed as a team project using a Raspberry Pi and environmental sensors. The project includes a multi-page website that documents the hardware setup, software implementation, and environmental monitoring results.

## Original Project

This project was originally developed as a team project as part of a course assignment.

Original repository:
https://github.com/Omega-oxide/Air-Quality

## Project Overview

This project monitors environmental air quality conditions using sensors connected to a Raspberry Pi. The system collects environmental data and presents project information through a website that documents the hardware setup, software implementation, and monitoring results.

## Technologies Used

- HTML
- CSS
- Python
- Raspberry Pi Zero 2 W
- Environmental air quality sensors
- GitHub Pages

## System Components

- Raspberry Pi Zero 2 W
- Environmental air quality sensors
- Python script for data collection
- Website interface for project documentation

## How to View the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `header.html` in a web browser.

The website will display the project pages describing the air quality monitoring system, hardware setup, and results.

## Project Demonstration

The website interface documents the air quality monitoring system, including the hardware setup, software implementation, and monitoring results. The system was tested using a Raspberry Pi Zero 2 W connected to environmental sensors that collected air quality data.

The project website provides multiple pages explaining:
- Hardware setup and components
- Software used to collect environmental data
- Documentation and project results
